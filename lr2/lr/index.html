Лабораторная работа 2

Описание

Веб-приложение для управления научными конференциями. Оно позволяет пользователям регистрироваться на конференции, указывать тему своих докладов, просматривать участников и отзывы, а также оставлять свои отзывы о конференциях. Пользователи могут редактировать свои регистрации, включая изменение темы доклада.

Функциональные возможности

Просмотр доступных конференций

Пользователь может просматривать список доступных конференций, видеть их название, место проведения, даты, темы, описание и условия участия.

Регистрация на конференцию с указанием темы доклада

Зарегистрированные пользователи могут выбрать конференцию и указать тему своего доклада при регистрации.

Просмотр и редактирование регистраций

Пользователь может просмотреть список своих регистраций на странице "Мои регистрации", редактировать тему доклада или удалить свою регистрацию.

Оставление отзывов о конференции

Пользователи могут оставлять отзывы о конференции и указывать рейтинг. Все отзывы видны на странице конференции.

Основные файлы и структура проекта

  • models.py — содержит модели Conference и Registration. Registration включает поле topic для указания темы доклада.
  • forms.py — содержит формы RegistrationForm для регистрации и RegistrationEditForm для редактирования темы доклада.
  • views.py — реализует основные представления для регистрации, редактирования, просмотра списка конференций и отзывов.
  • urls.py — определяет маршруты для всех страниц приложения.
  • templates/ — папка с HTML-шаблонами для страниц приложения.

Шаблоны

  • base.html
    Базовый шаблон с навигационным меню. Содержит ссылки на страницу с доступными конференциями, страницу регистрации, список регистраций пользователя и выход из системы.

  • register_for_conference.html
    Форма регистрации на конференцию, где пользователь указывает тему своего доклада.

  • edit_registration.html
    Форма редактирования регистрации, позволяющая изменить тему доклада.

  • conference_details.html
    Страница с деталями конференции, включая описание, отзывы, участников и условия участия.

  • my_registrations.html
    Страница со списком всех регистраций пользователя, где также отображаются темы докладов и доступны ссылки на редактирование и удаление регистраций.

URL маршруты

  • / — главная страница с навигацией.
  • /register_for_conference// — регистрация на конференцию.
  • /my_registrations/ — список регистраций пользователя.
  • /registration//edit/ — редактирование темы доклада.
  • /conference// — просмотр информации о конференции и оставленных отзывах.

Примечания

  • Для входа и выхода из системы используются стандартные Django views, подключенные через django.contrib.auth.urls.
  • В проекте используется условный рендеринг на основе статуса пользователя (user.is_authenticated) для отображения определенных ссылок в меню.
