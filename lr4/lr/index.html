Лабораторная работа 4

Описание проекта

Система управления отелем, включающая такие функции, как бронирование номеров, управление клиентами, сотрудниками, расписанием уборок и отчетами.

Стек технологий

  • Frontend: Vue.js 3, Pinia, Vuetify
  • Backend: Django (API)
  • База данных: PostgreSQL
  • Прочее: Axios, Material Design Icons

Основные компоненты и функциональность

1. Бронирование

  • Основной функционал:
  • Создание, редактирование и удаление бронирований.
  • Подключение клиента и комнаты к бронированию.
  • Указание дат заезда и выезда.
  • Интерфейс:
  • Список бронирований с возможностью редактирования и удаления.
  • Модальное окно для добавления/редактирования бронирования.

2. Управление клиентами

  • Основной функционал:
  • Добавление, редактирование и удаление клиентов.
  • Отображение списка клиентов с фильтрацией и поиском.
  • Интерфейс:
  • Таблица клиентов с колонками: имя, фамилия, город и последняя комната.
  • Модальное окно для добавления/редактирования клиента.

3. Управление сотрудниками

  • Основной функционал:
  • Добавление и редактирование информации о сотрудниках.
  • Отображение графика работы сотрудников.
  • Интерфейс:
  • Список сотрудников с кнопками для редактирования и удаления.

4. Уборка номеров

  • Основной функционал:
  • Создание и редактирование расписания уборок.
  • Назначение сотрудника на уборку определенного номера.
  • Интерфейс:
  • Таблица расписания уборок с фильтрацией по дате.

5. Отчеты

  • Основной функционал:
  • Генерация отчетов по бронированиям, клиентам и расписаниям.
  • Интерфейс:
  • Страница для настройки параметров отчетов и их скачивания.

Навигация

  • Главная страница: Информация о количестве свободных номеров и активных бронирований.
  • Клиенты: Список клиентов с кнопками "Добавить", "Редактировать", "Удалить".
  • Сотрудники: Управление списком сотрудников и их расписанием.
  • Номера: Список комнат, сгруппированных по этажам, с указанием доступности.
  • Расписание: Таблица графика уборок.
  • Бронирования: Отображение текущих бронирований с возможностью редактирования.
  • Отчеты: Генерация отчетов.
